## Deployment

To deploy this project run

1) Setup Api Key

2) Install Node

3) Install Dependencies

```bash
  npm install
```
4) Start backend

```bash
  cd backend
  node server.js
```
5) Start frontend

```bash
  cd frontend
  npm start
```
